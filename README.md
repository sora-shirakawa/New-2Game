# 錬金店経営ゲーム（3日プロトタイプ）

## 起動手順（Python不要）
### 方法A: ファイルを直接開く（最も簡単）
1. `index.html` をダブルクリックしてブラウザで開く
2. そのままプレイ可能（`localStorage` セーブ対応）

### 方法B: Node.js がある場合
1. `npm run start` を実行
2. ブラウザで `http://localhost:8000` を開く

## 構成
- `index.html`: 画面構造（タイトル/ゲーム/リザルト）
- `style.css`: UIスタイル
- `main.js`: ゲームロジック（曜日進行、依頼、納品、錬金、販売、セーブ）
- `package.json`: ローカル起動用スクリプト / buildチェック

## セーブ
- `localStorage` キー: `alchemy_shop_save_v1`
- タイトル画面「つづきから」でロード、リセットで削除

## ゲームクリアの目安
- 4週終了時に `3000G` 以上でクリア
- 月曜日は必ず素材依頼を行い、土日で高売価レシピ（魔力ポーション/万能薬）を優先すると達成しやすい
